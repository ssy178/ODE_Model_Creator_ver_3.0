================================================================================
KNOWN LIMITATIONS & NOTES
================================================================================

Limitations:
  - CAT reaction has no activator/inhibitor support (by design)
  - MMS, MMSF, MMFF require at least one activator (enforced with error)
  - MMSR, MMFR only support 3 parameters minimum
  - All inhibitor terms are multiplicative (no cooperative inhibition)

Design Decisions:
  - Followed original MATLAB structure exactly
  - Parameter names include species/modifier names for clarity
  - Activator dependencies encoded in Km parameter names
  - Error checking for required activators

Future Enhancements:
  - Could add cooperative binding terms
  - Could add Hill coefficient support
  - Could optimize parameter name generation
  - Could add XML import/export support
